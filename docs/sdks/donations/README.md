# Donations

## Overview

Crypto donation claim flow and supported currencies.

### Available Operations

* [claim](#claim) - Claim a crypto donation by tx hash + signed message
* [getAddresses](#getaddresses) - Get the treasury addresses to send crypto donations to
* [listSupportedCurrencies](#listsupportedcurrencies) - List cryptocurrencies accepted for donations (DB-backed; admin-managed)

## claim
