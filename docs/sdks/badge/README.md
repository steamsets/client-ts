# Badge
(*badge*)

## Overview

Any request that is related to steam badges, mostly badge search/tagging related.

### Available Operations

* [getTags](#gettags)
* [search](#search)
* [suggestSearch](#suggestsearch)
* [suggestTags](#suggesttags)
